Analysis Tools Button: The Analysis Tools button contains options such as Identify and Identify Buffer.

Application Menu Bar: The Application Menu Bar contains three main groupings of commands to help you conduct a Map search and to interact with the results.

Return to the top

Basemap Button: The Basemap button contains the different basemaps (e.g., Roads (Bing), Imagery (Bing), Topo (ESRI) and Roads (ESRI)) that can be used to change the map format.

Buffer Tool: The buffer tool allows you to select an area around a point, rectangle, circle, polygon or line on the lease displayed on the map, and to select information in the buffered area.

Return to the top

Data Grid: The Data Grid displays basic data about each search result.

Division of Interest (DOI): The tabulation of all interests in a particular property. Includes working, royalty, and overriding royalty interests and is usually expressed in decimals.

Drawing Tools Button: The Drawing Tools Button contains various drawing options such as, Draw a Point, Draw a Line, Draw Freeform and Draw Text.

Return to the top

Help Dropdown List: The Help dropdown list provides a list of the different ways you can receive assistance and report issues with CHK Search.

Return to the top

Legend: The Legend allows you to expand and collapse the layer tree using the two buttons at the top of the list. The Legend will only reflect what is visible on the map. As you zoom in and out on the map, the Legend adjusts accordingly.

Return to the top

Map Tab: The Map tab gives you access to a quick visualization of leases, wells, facilities, and landgrids.

My Properties: My Properties is a list you create to save the properties that you work with frequently.

Return to the top

Preview Pane: The Preview Pane contains detailed information on a selected result in the Data Grid.

Return to the top

Search Toolbar: The Search Toolbar is used to manage both ad hoc and saved searches, so that you can quickly setup for your next search.

Search Type “And”/Search Type “Or”: Search Type “And”/Search Type “Or allows you to search for leases within a particular time frame.

VAL: Visual Asset Locator

Visual Asset Locator: The Visual Asset Locator (VAL) is a web mapping application used by the Chesapeake Contact Center. This application provides a mapping interface to quickly navigate to a geographic location.
